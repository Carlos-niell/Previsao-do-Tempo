# 🌤️ Previsão do Tempo

Uma aplicação web moderna e responsiva desenvolvida para consultar e exibir informações meteorológicas em tempo real de qualquer cidade do mundo. O projeto consome dados diretamente da API do **OpenWeatherMap**.

---

## 🚀 Funcionalidades

- **Busca em Tempo Real:** Pesquisa a previsão meteorológica através do nome de qualquer cidade.
- **Dados Detalhados:** Exibe informações essenciais como:
  - Temperatura atual (ºC)
  - Condição climática (com ícones ilustrativos)
  - Humidade do ar
  - Velocidade do vento
- **Design Responsivo:** Interface fluida que se adapta perfeitamente a dispositivos móveis, tablets e computadores.
- **Tratamento de Erros:** Alertas amigáveis caso a cidade não seja encontrada ou ocorra alguma falha na requisição.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização moderna, layout responsivo e transições.
- **JavaScript (ES6+):** Manipulação assíncrona do DOM e consumo da API utilizando `fetch`.
- **API OpenWeatherMap:** Serviço externo utilizado para obter os dados climáticos em tempo real.
